# CNVRT document templates

These are controlled source documents for client-specific versions. Keep the master generic and create a new file for every client.

## Codex workflow

1. Download or attach the current master template.
2. Attach the relevant CRM record, discovery response and approved commercial terms.
3. Ask Codex to preserve the template's layout and replace only the client-specific content.
4. Require an editable DOCX and a visually checked PDF.
5. Upload the approved result in Admin → Documents and assign it to the client.

Never invent credentials, claims, prices, deliverables or dates. Any missing decision must remain clearly flagged for approval.
